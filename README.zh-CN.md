本项目是由`Windward`驱动，`Spring`作为DI容器，`MongoDB`作为持久化存储实现的一款基于`WebSocket`及`Protobuf`的
即时通讯服务端。目前相关功能正在开发中🐻‍❄️

## 快速启动

- 打包
  ```shell
    ./gradlew clean jar
  ```
- 启动
  ```shell
    java -jar build/libs/wim-0.1-SNAPSHOT.jar
  ```
- 完毕
